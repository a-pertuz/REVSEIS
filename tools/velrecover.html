A tool for interpolating 2D seismic velocity data from sparse velocity analysis found in seismic sections. VELRecover helps geophysicists and seismic interpreters to create complete velocity models from sparse velocity picks.

GitHub Repository Features Troubleshooting Integration Guide

Features

  • Velocity Data Interpolation - Create complete velocity models from sparse velocity picks
  • Interactive Visualization - View velocity distributions in real-time
  • Smoothing Tools - Apply Gaussian blur to create smoother velocity models
  • Multiple Export Formats - Save interpolated data in text or binary format
  • SEGY Integration - Load SEGY files to provide spatial context for interpolation
  • Customizable Workflows - Tweak interpolation parameters to match your data

Installation

Requirements

  • Python 3.12 or higher
  • Required packages: PySide6, NumPy, pandas, appdirs, matplotlib, seisio

Using pip

pip install velrecover

From Source

pip install git+https://github.com/a-pertuz/velrecover.git

Launching VELRecover

After installation, launch the application by typing:

velrecover

Quick Start Guide

Folder Structure

VELRecover organizes data in the following folder structure:

  • SEGY/ - Store seismic SEGY files
  • VELS/RAW/ - Store input velocity data files
  • VELS/2D/ - Store interpolated velocity models

Step-by-Step Workflow

Load Velocity Data: Click "Load Text File" and select a velocity text file with CDP, TWT, Velocity columns
Load SEGY File: Click "Load SEGY File" to provide spatial context for interpolation
Run Interpolation: Click "Interpolate" to process the data and fill gaps between sparse velocity picks
Apply Smoothing (Optional): Enter a Gaussian blur value and click "Apply Gaussian Blur"
Export Results: Save your data in text or binary format using the export buttons

Tip: Binary format (.bin) is suitable for migration in Seismic Unix, making it compatible with InSeis workflows.
