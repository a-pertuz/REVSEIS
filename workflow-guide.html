Workflow Implementation Guide

This guide demonstrates how to implement REVISEIS tools in your seismic data recovery workflows. Learn how to take your vintage seismic data from paper to modern digital formats ready for interpretation.

The REVISEIS workflow consists of three main stages, each handled by a dedicated tool:

Stage 1: Digitization with SEGYRecover

Convert paper seismic sections or raster images to standard SEGY format, preserving amplitude and spatial information.

Input: Scanned Image
→
Output: Digital SEGY

Stage 2: Velocity Modeling with VELRecover

Create complete velocity fields from sparse velocity picks using interpolation algorithms.

Input: Sparse Points
→
Output: Complete Model

Stage 3: Processing with InSeis

Apply Seismic Unix processing algorithms to enhance, filter, and analyze the digitized data.

Input: Raw Data
→
Output: Processed Data
