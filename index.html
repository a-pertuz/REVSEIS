Welcome to REV-SEIS

REV-SEIS helps geologists and geophysicists breathe new life into old seismic data originally recorded on paper. With SEGYRecover you can digitize seismic sections into SEG-Y format, VelRecover allows you to interpolate sparse velocity picks, and InSeis helps enhance vintage seismic data through Seismic Unix processes, making it all accessible for modern interpretation.

REV-SEIS Tools

Each tool in the REV-SEIS suite addresses a specific challenge in the vintage seismic data recovery process:

SEGYRecover

Python Windows

A tool for digitizing scanned seismic sections and converting them to standard SEGY format.

Learn more

VelRecover

Python Windows

A tool for interpolating 2D seismic velocity data from sparse velocity analysis found in seismic sections.

Learn more

InSeis

Python Windows WSL

A graphical interface for creating and running Seismic Unix workflows on Windows through WSL.

Learn more

    Installation Guide

    New to command line?
    Don't worry! We'll explain each step. The command line is just a text interface where you type commands instead of clicking buttons.

    Step 1: Install Python

    Python is the programming language that powers our tools. You need to install it first:

    1. Go to python.org and click the "Download Python" button
    2. Run the downloaded file
    3. Make sure to check the box that says "Add Python to PATH" during installation
    What does "Add Python to PATH" do?
    It allows you to run Python commands from any folder on your computer. This is essential for our tools to work properly.

    Step 2: Installing REVSEIS

    Open Command Prompt (search "cmd" in Windows Start menu) and run each command to install the tools:

    pip install segyrecover
    pip install velrecover
    pip install inseis

    Alternative: Install from GitHub

    For the latest development version:

    pip install git+https://github.com/a-pertuz/segyrecover.git
    pip install git+https://github.com/a-pertuz/velrecover.git
    pip install git+https://github.com/a-pertuz/inseis.git
    The git+ prefix instructs pip to install directly from the GitHub repository's latest version.

    Step 3: Launch the Tools

    Open Command Prompt and run any of these commands:

    segyrecover
    velrecover
    inseis

    On first run, each tool will guide you through selecting a data directory.

    Step 4: Install WSL (for InSeis only)

    If you plan to use InSeis, you need to set up Windows Subsystem for Linux:

    1. Open Command Prompt as Administrator (right-click, select "Run as administrator")
    2. Run the following command:
    wsl --install

    After installation completes, restart your computer.

    Step 5: Install Seismic Unix (for InSeis only)

    After restarting and if WSL is properly installed:

    1. Open WSL (search for "wsl" or "ubuntu" in Windows Start menu)
    2. Copy and paste this entire command:
    bash -c "$(wget -qO- https://gist.githubusercontent.com/a-pertuz/2b341bc8af2a37cde820d829f2789d99/raw/install_seismic_unix.sh)"

    This script will automatically install Seismic Unix. The process may take several minutes.

    Important: During installation accept Seismic Unix license agreement when prompted.
    If you don't accept, the installation will fail and you won't be able to use InSeis.
