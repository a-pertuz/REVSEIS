Overview

This documentation hub provides installation guides and essential information for three specialized seismic data processing tools. Each tool addresses a specific need in seismic data workflows:

SEGYRecover

Python Windows 1.0.0 GPL-3.0

A tool for digitizing scanned seismic sections and converting them to standard SEGY format.

Learn more

VELRecover

Python Windows 1.0.0 GPL-3.0

A tool for interpolating 2D seismic velocity data from sparse velocity analysis found in seismic sections.

Learn more

InSeis

Python Windows WSL 1.0.0 GPL-3.0

A graphical interface for creating and running Seismic Unix workflows on Windows through WSL.

Learn more

Quick Installation Guide

Step 1: Install Python

Download Python from python.org and select "Add Python to PATH" during installation.

Step 2: Install the Tools

pip install segyrecover velrecover inseis
Install all tools at once with a single command

Or install them individually:

pip install segyrecover
pip install velrecover
pip install inseis

Step 3: Launch

Run each tool by typing its name in Command Prompt:

segyrecover velrecover inseis
